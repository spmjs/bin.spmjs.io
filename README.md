# spmjs.bin

## Install

### Get Source Code

```bash
$ git clone git://github.com/spmjs/bin.spmjs.io.git
$ cd spmjs.io
$ npm install
```

### Sync

```bash
$ ./bin/sync --lastest
$ ./bin/sync --all
```

### Start devserver

```bash
# server
$ npm run startdev
# assets live rebuild
$ amb build -w
```
